```bash
src/
```

### 项目功能

- [x] 自动打开浏览器
- [x] 自动登录账号
- [x] 自动转发
- [x] 自动点赞
- [x] 自动书签

  - 自动评论

### 环境变量

```bash
Auth_Token= 账号token, 多个账号用@隔开
Url= 需要冲的链接
```

- 安装node环境 [https://www.cnblogs.com/xiaodaidaiPro/p/18253830](https://www.cnblogs.com/xiaodaidaiPro/p/18253830)
- 安装依赖

```bash
npm install
```

- 运行项目

```bash
npm run start
```
